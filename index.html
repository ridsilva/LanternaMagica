Welcome lanternist!

To start your journey, select one of the available slides to be painted.

Click on each slide to view it more closely, and then click "Chose Slide" to finalize your choice and begin working.

This icon allows the user to choose a palette to paint with. If in the middle of the painting another palette is chosen all progress will be lost.

This icon allows the user to undo their last modification. It is only possible to perform x undo's

This icon allows the user to redo what was undone. It is only possible to redo if an undo had been previously been done.

Clicking in "Final" will animate the painting.

If you press the "s" key before clicking "Final," only an image will be saved. After clicking "Final," a GIF will be saved.

You are able to mix colors in the fill option.
